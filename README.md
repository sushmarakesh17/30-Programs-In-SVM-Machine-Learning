## 📌 Project Overview

This project demonstrates **30 Support Vector Machine (SVM) programs** along with an end-to-end Machine Learning application.

The deployed application is a **Student Performance Prediction System** that predicts whether a student will **Pass or Fail** using an SVM classification model.

---

## 🎯 Objectives

- Implement different SVM concepts
- Understand SVM classification and regression
- Apply preprocessing and feature scaling
- Train and deploy a Machine Learning model using Streamlit

---

## 🤖 Algorithm Used

**Support Vector Machine (SVM)**

SVM is a supervised Machine Learning algorithm used for classification and regression by finding the optimal decision boundary between classes.

---

## 📚 SVM Concepts Covered

- Linear Kernel SVM
- RBF Kernel SVM
- Polynomial Kernel SVM
- Sigmoid Kernel SVM
- Classification using SVM
- Regression using SVR
- Feature Scaling
- Hyperparameter Tuning
- Model Evaluation
- Model Deployment

---

### Prediction:

- Pass
